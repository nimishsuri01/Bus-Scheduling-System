# Bus Scheduling System

A React + Vite bus scheduling dashboard with live route tracking, fleet management, and reservation features.

## Run locally

```bash
npm install
npm run dev
```

Then open:

```text
http://localhost:3000
```

## Preview production build

```bash
npm run build
npm run preview -- --host 0.0.0.0 --port 3000
```

## Deploy easily

This project is ready for simple hosting on:
- Vercel
- Netlify
- GitHub Pages

For Vercel or Netlify, connect the repository and use the default Vite build settings.
